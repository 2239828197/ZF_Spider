# ZF_Spider

A python program can be used to login ZF and select lessons.

这是一个用Python写的正方教务抢课脚本，由于每个学校的教务系统可能存在改动，所以不一定可用。

使用方法：
config.json配置文件中填写教务系统的url，学号与密码即可运行main.py

暂时只支持公选课抢课
